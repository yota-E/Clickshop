Politique de confidentialité

Chez ClickShop, nous accordons une grande importance à la confidentialité de vos données personnelles. Cette page explique quelles données nous collectons, comment nous les utilisons, et vos droits.

1. Données collectées

  • Nom et prénom
  • Adresse email
  • Mot de passe (crypté)
  • Données de navigation sur le site
  • Messages envoyés via les formulaires

2. Utilisation des données

Vos données sont utilisées uniquement pour :

  • Gérer votre compte utilisateur
  • Gérer vos commandes et votre panier
  • Vous contacter si besoin (support, confirmation, etc.)
  • Améliorer l’expérience sur le site

3. Conservation des données

Vos données sont conservées aussi longtemps que votre compte est actif. Vous pouvez demander la suppression de votre compte à tout moment.

4. Partage des données

Nous ne partageons jamais vos données personnelles avec des tiers sans votre consentement.

5. Vos droits

Vous pouvez à tout moment :

  • Accéder à vos données
  • Les modifier
  • Demander leur suppression

Pour exercer ces droits, contactez-nous à : killolocoeurcy@gmail.com
